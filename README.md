# Demo Website Starter

## ⚡️ Quick Start

```bash
yarn create next-app -e https://github.com/Joshhortt/demo-website-starter
# or
npx create-next-app -e https://github.com/Joshhortt/demo-website-starter
```

### Install Dependencies

```bash
yarn install
# or
npm install
```

### Run Localhost

```bash
yarn dev
# or
npm run dev
```
